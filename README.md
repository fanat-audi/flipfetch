## Установка:
```bash
curl -fsSL https://raw.githubusercontent.com/fanat-audi/flipfetch/main/install.sh | sh
```
---
### Запуск:
```bash
flipfetch
```
---

## Примечание (Linux):
Если выдает ошибку прав порта, добавьте себя в группу `dialout` и перезагрузитесь:
```bash
sudo usermod -aG dialout $USER
```
